# Журнал прогресса

Здесь складываются ежедневные заметки о работе. Один файл на день: `ГГГГ-ММ-ДД.md`.

Каждый день в 18:00 запускается `save-progress.sh`, который:

1. добавляет в файл сегодняшнего дня снимок изменённых файлов;
2. коммитит все изменения по всем проектам (Framy, Claude Brain, PlayID) в один репозиторий;
3. пушит на GitHub (когда настроен remote) или напоминает запустить `git push` вручную.

Чтобы записать содержательную заметку за день, просто отредактируй раздел **Заметки**
в файле текущего дня — или попроси меня это сделать.
