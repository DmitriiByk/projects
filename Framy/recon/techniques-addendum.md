# Каталог приёмов — добавления (T16–T17 + «две школы»)

Расширение `build-game-ui/references/techniques.md`. Выведено из анализа Dragonia (раскрашенный
ForgeX) против Vegas Hero. При установке — дописать в `techniques.md`.

## Две школы декора (рамка выбора приёма)
Один и тот же контракт **3-слайс `part=Left/Middle/right`** реализуют двумя путями. `mapStrategy`
выбирает по сигналу профиля:
- **Растровая нарезка (baked)** — куски картинки `IMAGE scaleMode=CROP`, уложенные L/M/R. Сигнал:
  IMAGE-CROP в part-нарезке, мало векторов/градиентов. → приём **T16**, и реюз ассета, НЕ перерисовка.
- **Процедурный вектор/эффекты** — вектор-полигоны, opacity-стек, градиенты, тайл-текстура. Сигнал:
  VECTOR-полигоны (часто polygon/скос), стек слоёв с разной opacity, многостоповые градиенты. →
  приёмы T1/T7/T8/T9-skew + урок L6.

## T16 — Растровый 9-slice фрейм/кнопка/фон (baked art)
- **Когда:** бренд держит декор в готовых ассетах (школа Dragonia): рамки карточек, кнопки, фоны.
- **Структура:** контейнер L / Middle / R (auto-layout, центр FILL); в каждом — один или несколько
  `RECTANGLE` с `IMAGE`-заливкой `scaleMode=CROP` (куски рамки). Прозрачный `RECTANGLE` «Space»
  (opacity 0) = слот под контент/арт (негативное пространство). Тянется центральным куском.
- **Опц.:** доп. полноразмерный `IMAGE scaleMode=FILL` слой «Border» поверх (Frame-Sport).
- **Реюз > имитация:** перерисовать нарезку вектором нельзя достоверно (это baked-арт) — переиспользуй
  компонент-рамку (`Frame-gamecard`/`Frame-Sport`) инстансом, контент клади в слот.
- **Распознать (dumpSignals):** `IMAGE scaleMode=CROP` в нодах с именами L/Middle/R/part + прозрачный
  «Space». Флаг-кандидат `is_raster_slice`.
- **Регресс:** Dragonia `Frame-gamecard` (28671:59037), `Frame-Sport` (28671:133390),
  `BG-casino-categories` (30028:294916), Button parts (77:35600 → Left/Middle/right = IMAGE CROP).

## T17 — ArtGuide / safe-zone overlay
- **Когда:** баннеры/герои с hero-артом, который нельзя двигать на текст/кнопки.
- **Что:** компонент-оверлей в арт-слоте, размечающий зоны (`Guide.Area` `variant=danger/warning/
  default`) — куда арт можно/нельзя. Дизайн-ops направляющая, не визуальный финиш (в проде скрыта).
- **Реюз:** держать отдельным компонентом, класть в Art-слот баннера рядом с картинкой.
- **Регресс:** Dragonia MainBanner → MainBanner.Art → ArtGuide (Guide.Area ×4).

## Доп. наблюдение (brand-приём Dragonia)
- **Золотой градиент-лейбл:** текст/иконка с `GRADIENT_LINEAR` бело→`#EFCCAB` + тёмная SOLID-обводка +
  hard `DROP_SHADOW` (radius 0). Это T12 с конкретными brand-значениями.
