## Язык декора Dragonia (вывод)
- **Главные поверхности** (Button, баннеры, ProgressBar) → 3-слайс/растровая база + **градиент** +
  **тень-отрыв** (объём «выдавлен из фона»).
- **Поля ввода** (Input) → **inner-shadow** (вдавленность), инверсия объёма кнопок.
- **Мелкие чипы/коллауты** → остаются плоскими (де-акцент = только палитра).
- Структурный приём бренда — **3-слайс через `part=Left/Middle/right`** (T1), металл-градиент (T8),
  тень (T11). Текстура/скос (T7/T9-skew) — у Vegas Hero, у Dragonia их в этих компонентах нет.

## Оговорки (как просил пользователь)
- Часть страниц концепта удалена/изменена: **GameBanner отсутствует**; Chip/MainBanner/GameCard имеют

- Баннеры сняты shallow (d≤2) из-за тяжести — дельта на уровне фон/материал, без глубокого стека.
