# Projects

Единый репозиторий для ежедневного сохранения прогресса по проектам. Позволяет
подгружать работу на другом устройстве и делиться с коллегами через GitHub.

## Структура

- **Framy/** — код-проект (дизайн-система, токены, пайплайны).
- **Claude Brain/** — заметки, профили, рабочие воркфлоу.
- **PlayID/** — проект PlayID.
- **PROGRESS/** — журнал прогресса по дням (`ГГГГ-ММ-ДД.md`).
- **save-progress.sh** — скрипт ежедневного коммита (запускается в 18:00).

## Ежедневное сохранение

Скрипт `save-progress.sh` собирает все изменения в один коммит и пушит на GitHub.
Запуск настроен на 18:00. Вручную:

```bash
bash save-progress.sh ["сообщение коммита"]
```

## Загрузить на другом устройстве

```bash
git clone <url-репозитория>
```

## История Framy

Прежняя отдельная git-история Framy сохранена в резервной копии
(`framy-git-backup-*.tar.gz`) и в `Framy/.git_off/`. В общий репозиторий
файлы Framy вошли как обычные, без старой истории.
